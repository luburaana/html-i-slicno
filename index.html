Linijski CSS
ovo se ne dira

*Kao što već znamo, linijski CSS predstavlja samo stilski dodatak HTML jeziku. To nam govori da je sintaksa za tabelu nepromijenjena u odnosu na HTML. Jedino što se mijenja jeste kako mi tu tabelu stilizujemo.

*Prvo ćemo se podsjetiti sintakse za ubacivanje klasične 2x2 tabele u našu stranicu.

*Sada (stavku po stavku) da se podsjetimo šta koji od ovih tagova označava:

  • Table border - daje vizualnu strukturu tabeli, debljine 1, suštinski je odvaja od pozadine i razgraničava
  • tr tag - tr je skraćeno za table row, ubacujemo red u tabeli
  • td tag - td je skraćeno za table data, u suštini samo oznaka za ćeliju

*Ovdje dolazimo do dijela gdje nam linijski CSS pomaže. Kao što sam već napomenula, njegov zadatak je da stilizuje našu tabelu. Dakle, navešćemo tagove za stil:

  • (Table) border - daje vizualnu strukturu tabeli, debljine 1, suštinski je odvaja od pozadine i razgraničava, dajemo joj brojčanu vrijednost (1,2,3..)
  • Table background color - ubacujemo ga uz style tag kao kod DIV i SPAN, boju biramo ili riječima ili HEX kodom
  • th tag - th je skraćeno za table header, daje naslov redu/koloni, zavisi gdje unosimo
  • border collapse-daje efekat spojenih ivica ćelija; nije nužno samo je estetski dodatak
